# DocLink
This repository is a study-focused project with the intent to practice and update my knowledge in new technologies

The first and main objective of this project is to provide a service that can be able to consume documents from a specific file directory, the main target extension is PDF, the code will read it, extract the document data, send it to a database and keep it available to future access through the application's front-end.

The first step was to chose what kind of architecture would be used for this code. Naturally, Domain-Driven Deveolpment was the one selected.
