Este repositório é um projeto voltado para o estudo, com o objetivo de praticar e atualizar meus conhecimentos em novas tecnologias

O primeiro e principal objetivo deste projeto é fornecer um serviço capaz de processar documentos de um diretório específico, a extensão principalmente trabalhada será PDF. O código irá ler o documento, extrair seus dados, enviá-los para um banco de dados e mantê-los disponíveis para acesso futuro por meio do front-end do aplicativo.

O primeiro passo foi escolher que tipo de arquitetura seria utilizada para este código. Naturalmente, o Desenvolvimento Orientado a Domínio foi o escolhido.
